Git

  • git init- создание нового репозитория в пустой директории.
  • git clone - клонирование репозитория из удаленного хранилища на локольную машину.
  • git push- выгрузка изменений в удаленный репозиторий.
  • git pull- получение изменений из удаленного репозитория на локальную машину.
  • git status- просмотр списка измененных файлов.
  • git branch- создание ветки.
  • git checkout- переход в ветку.
  • git commit- сохранение изменений.
  • git log- просмотр истории коммитов.
